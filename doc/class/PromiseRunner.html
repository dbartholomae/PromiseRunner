Class: PromiseRunner

Defined in: src\PromiseRunner.coffee

Overview

A PromiseRunner runs multiple functions that are dependent on each other in the correct order. Tasks are defined as an object { taskName: task, ... }, with task being either a function that has no dependencies, an array where the first element is the function and the rest are the names of tasks it depends on, or an object { fn: function, deps: [string] }. Each task will be run with the results of its dependencies as parameters. Tasks can either return values or promises that return their values.

Instance Method Summary

  • # (Promise) run(tasks) Run a set of tasks and return a promise that resolves when all are run and rejects with "Dependency Cycle Found" if there is a dependency cycle.
